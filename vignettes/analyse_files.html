Extracting all file parameters is easy:

csv_filename <- "collected_parameters.csv"
if (!file.exists(csv_filename)) {
  folder <- "/home/p230198/Peregrine"
  all_parameter_filenames <- paste(folder, list.files(folder, pattern = "\\.RDa"), sep = "/")
  df <- collect_parameters(all_parameter_filenames, verbose = TRUE)
  write.csv(
    x = df,
    file = csv_filename,
    row.names = TRUE
  )
}

Note that the comma-seperated file is only created when absent, as this is a long step.

If an incipient species tree is simulated for a longer time, I expect it to have more branches, thus making the next steps harder, thus causing a BEAST2 run to take too long.

Let’s check this:

ggplot2::ggplot(data = df, ggplot2::aes(df$age)) + ggplot2::geom_histogram(binwidth = 1)

Age appears not to be very predictive if a simulation will succeed.
