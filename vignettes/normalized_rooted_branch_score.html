normalized_rooted_branch_score

Femke Thon

9 May 2016

  library(Cer2016)

Description

Calculate the normalized rooted branch score of trees. This function compares pairs of trees.

Usage

normalized_rooted_branch_score(filenames)

Arguments

The argument is a list of files that have to be compared to each other (filenames) of an arbitrary length.
